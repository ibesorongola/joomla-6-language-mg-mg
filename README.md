# Joomla Language Pack mg-MG (Malagasy)

Ce dépôt génère automatiquement le pack de langue malgache pour Joomla 6 à partir du pack officiel français.

- **Dernière version** : voir [Releases](https://github.com/ibesorongola/joomla-6-language-mg-mg/releases)
- **Installation** : utilisez l'URL du ZIP dans Joomla (Système → Installer → depuis une URL) ou téléchargez-le et installez-le manuellement.
- **Mise à jour** : une fois installé, le serveur de mise à jour est automatiquement configuré. Les nouvelles versions seront proposées dans le gestionnaire de mise à jour de Joomla.
